从0到1使用uniapp写小程序(第二章：引入uview)

July 26, 2023 One-minute read
howey
前端
uniapp • vue

安装 scss Link to heading

uview 依赖于 scss 模块

安装 view Link to heading

下载安装: 登录一下 hbuild，一键导入 huilder 插件市场-view

配置 Link to heading

项目 main.js 中引入： 

// main.js
import uView from "@/uni_modules/uview-ui";
Vue.use(uView);

注意这两行要放在 import Vue 之后。

引入 scss Link to heading

@import "@/uni_modules/uview-ui/theme.scss";

app.vue 引入 基础样式 首行加入 lang=“scss” Link to heading

<style lang="scss">
	/* 注意要写在第一行，同时给style标签加入lang="scss"属性 */
	@import "@/uni_modules/uview-ui/index.scss";
</style>

配置 easycom 组件模式 Link to heading

打开 pages.json 加入 

// pages.json
{
  // 如果您是通过uni_modules形式引入uView，可以忽略此配置
  "easycom": {
    "^u-(.*)": "@/uni_modules/uview-ui/components/u-$1/u-$1.vue"
  },

  // 此为本身已有的内容
  "pages": [
    // ......
  ]
}
