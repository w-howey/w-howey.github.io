一、搭建 centos 虚拟机， Link to heading

1.分区等设置 Link to heading

下载镜像导入 ，分区： 硬件：内存 2G，硬盘 50G 分区：/boot 挂载 1g /swap 挂载 4G （内存不够可采用磁盘当内存使用） /挂载 45G

2.网络 ip 设置 Link to heading

（1）虚拟网络编辑器设置：选择 vmnet8 子网 ip 设置网段修改为任意网段 例如 10

nat 设置：原则同网段就想

（2）本地机设置：vmnet8 ipv4 属性 （3）虚拟机 centos 系统设置：

vim /etc/sysconfig/network-scripts/ifcfg-ens33

如下： 最后重启一下虚拟机。。 ping 一下外网和内网，查看是否能通信，通信就搞定 设置一下 host vim /etc/hosts

安装一下软件工具包 yum install -y epel-release 关闭防火墙 systemctl stop firewalld 禁止防火墙开机自启 systemctl disable firewalld.service

番外：linux 的权限设置 /etc/sudoers cd /opt/ sudo rm -rf rh sudo mkdir module sudo mkdir software sudo chown howey:howey module/ software/

二、安装 jdk Link to heading

1.卸载自带的 jdk Link to heading

rpm -qa |grep -i java |xargs -n1 rpm -e --nodeps rpm -qa：查询所安装的所有 rpm 软件包 grep -i ：忽略大小写 xargs -n1 ：表示每次只传递一个参数 rpm -e -nodeps ：强制卸载软件

2.下载 jdk for linux sftp Link to heading

上传到目录/opt/software/ 解压安装 tar -zxvf jdk-8u331-linux-x64.tar.gz -C /opt/module/

3 配置系统变量 Link to heading

sudo vim /etc/profile.d/my_env.sh 重新加载资源 source /etc/profile java -version

2.解压安装 Link to heading

tar -zxvf hadoop-3.3.1.tar.gz -C /opt/module/
